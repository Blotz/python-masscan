# python-masscan

python-masscan is a python library which helps in using masscan port scanner. 

## INSTALL

```bash
pip install python-masscan
```

## USAGE

```python

import masscan


mas = masscan.PortScanner()
mas.scan('172.0.8.78/24', ports='22,80,8080')
print mas.scan_result

```
